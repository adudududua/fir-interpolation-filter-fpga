# 高阶数字插值滤波器设计与验证

## 1. 项目简介

本项目面向“数字设计域：高阶数字插值滤波器设计与验证”赛题，完成了适用于高保真音频重构场景的 FPGA 原型系统设计。系统围绕 24 bit 有符号音频采样信号，完成 MATLAB 建模、RTL 编码、功能仿真、板级验证、资源利用率评估、功耗评估、时序分析以及结构 / 字长优化。

项目核心目标是在 FPGA 内部实现多倍率 FIR 插值滤波链路，支持 44.1 kHz / 48 kHz 两类音频采样率家族，并输出 4×、8×、128× 三档插值结果。板级主验证采用 AD9708 高速 DAC 与示波器完成；辅助听感展示采用 VS1053 音频解码模块完成 A/B 对比播放，用于直观展示保持重构与 FIR 插值重构之间的听感差异。

在基准版本基础上，本项目进一步完成了后级 2× FIR 字长优化、累加器位宽优化以及前级 4× FIR 稀疏裁剪可行性分析。最终优化版本在保持板级功能正常和时序满足的前提下，将 Slice LUTs 从 17572 降低至 14852，将 DSP48E1 从 6 个降低至 1 个，将 CARRY4 从 4043 降低至 3383。

## 3. 系统总体结构

本项目采用“两条验证线路”：

```text
线路 1：主验证线路
正弦 ROM / 音频 PCM ROM
        ↓
24 bit signed 输入采样
        ↓
FPGA FIR 插值链
        ↓
4× / 8× / 128× 插值输出
        ↓
AD9708 高速 DAC
        ↓
示波器观察 dac_clk 与模拟输出波形

线路 2：辅助听感线路
48 kHz / 24 bit signed 音频 ROM
        ↓
FPGA 内部 A/B 实时处理
        ├─ A 路：保持重构
        └─ B 路：FIR 插值重构
        ↓
VS1053 播放 48 kHz / 16 bit PCM
        ↓
耳机 / 喇叭听感对比
```

其中 AD9708 + 示波器为赛题主验证支路；VS1053 为辅助听感展示支路，不作为 4× / 8× / 128× 高采样率输出的主验证证据。

### 3.1 FIR 插值链路

主验证支路采用 4× FIR 与多个 2× FIR 级联的多级插值结构：

```text
24 bit signed 输入
        ↓
4× FIR 插值
        ↓
8× 输出 / 继续插值
        ↓
2× FIR 级联
        ↓
16× / 32× / 64× / 128×
        ↓
AD9708 DAC 输出
```

该版本在基准版本基础上完成后级 2× FIR 字长优化和累加器位宽优化，是本项目创新指标对应的最终优化版本。

|---|---:|---:|---:|---:|
| 4× FIR | 18 bit | Q16 | 52 bit | 155 |
| 2× FIR | 14 bit | Q12 | 45 bit | 29 |

### 6.4 `vs1053_fpga_48k_interp_ab_ok.bit`

用途：辅助听感展示。

该版本读取 48 kHz / 24 bit signed 音频 ROM，在 FPGA 内部实时生成两路音频：

- `sw0 = 0`：A 路，保持重构，声音更粗糙、更尖锐；
- `sw0 = 1`：B 路，FIR 插值重构，声音更平滑、更柔和。

VS1053 只负责播放 FPGA 输出的 48 kHz / 16 bit PCM 音频流，A/B 差异由 FPGA 内部实时产生。

## 10. 优化版本与创新指标完成情况

### 10.1 优化路线

本项目围绕赛题创新指标完成了三轮优化探索：

| 优化方向 | 结果 | 是否用于最终 RTL |
|---|---|---|
| 后级 2× FIR 字长优化 | 成功，18 bit Q16 → 14 bit Q12 | 是 |
| FIR 累加器位宽优化 | 成功，4× ACC 56 → 52，2× ACC 48 → 45 | 是 |
| 4× FIR 固定 155 tap 小系数稀疏裁剪 | 不适合，`prune_thr=1` 即不满足指标 | 否 |

最终采用 `acc_opt` 版本作为优化版工程。

---

### 10.2 创新指标 1：结构 / 算法优化

#### 10.2.1 分级插值结构

本项目没有采用单级直接 128× 插值结构，而是采用 4× FIR 与多个 2× FIR 级联的多级插值方案。该结构将高倍率插值任务分解为多个较低复杂度的插值阶段，使系统能够灵活支持 4×、8×、128× 多倍率输出。

#### 10.2.2 基于频率选择性的分级优化策略

不同插值级的滤波压力并不相同：

- 4× FIR 前级需要完成 44.1 kHz / 48 kHz 输入音频到 176.4 kHz / 192 kHz 的关键插值滤波，过渡带较窄，因此对滤波器阶数和系数精度要求较高。
- 后级 2× FIR 的输入采样率已经较高，过渡带明显更宽，因此可以采用更低的系数字长和更小的累加器位宽。

因此，优化版采用“前级保持高精度，后级压缩字长”的分级结构优化策略：

```text
4× 前级 FIR：
    COEFF_W = 18
    FRAC_W  = 16
    ACC_W   = 52
    NTAPS   = 155

后级 2× FIR：
    COEFF_W = 14
    FRAC_W  = 12
    ACC_W   = 45
    NTAPS   = 29
```

这种结构优化避免了所有滤波级统一使用保守高字长参数，降低了后级重复插值链路的硬件资源开销。

---

### 10.3 创新指标 2：字长优化策略

#### 10.3.1 后级 2× FIR 字长搜索

本项目针对后级 2× FIR 建立了 MATLAB 定点字长搜索流程，对 `COEFF_W`、`FRAC_W` 和小系数裁剪阈值进行联合搜索。搜索对象包括：

```text
FRAC_W = 16 / 15 / 14 / 13 / 12
COEFF_W = FRAC_W + 2
PRUNE_THR = 0 / 1 / 2 / 4 / 8 / 16
```

最终后级 2× FIR 选取：

```text
COEFF_W = 14
FRAC_W  = 12
ACC_W   = 45
NTAPS   = 29
```

#### 10.3.2 后级 2× FIR 优化后频响性能

优化后的 2× FIR 同时适用于 176.4 kHz → 352.8 kHz 和 192 kHz → 384 kHz 两种模式，MATLAB 验证结果如下：

| 模式 | 通带峰峰纹波 | 通带 ±纹波 | 阻带衰减 | 群延迟 | 是否满足 |
|---|---:|---:|---:|---:|---|
| 176.4 kHz → 352.8 kHz | 0.00201752 dB | 0.00100876 dB | 80.42743153 dB | 14 samples | 是 |
| 192 kHz → 384 kHz | 0.00132787 dB | 0.00066393 dB | 83.21212199 dB | 14 samples | 是 |

该结果说明，后级 2× FIR 即使从 18 bit Q16 降低到 14 bit Q12，仍能满足通带纹波、阻带衰减和严格线性相位要求。

#### 10.3.3 累加器位宽优化

在完成后级 2× FIR 系数字长优化后，本项目进一步对 FIR 内部累加器位宽进行压缩：

```text
4× FIR：ACC_W = 56 → 52
2× FIR：ACC_W = 48 → 45
```

该优化不改变 FIR 系数和频率响应，只减少内部乘加链路、舍入饱和模块和比较逻辑的数据通路宽度。实现结果表明，累加器位宽优化后系统仍然满足时序要求，并能够正常完成板级 DAC 输出。

#### 10.3.4 4× FIR 稀疏裁剪可行性分析

为了进一步考察前级 4× FIR 是否存在稀疏化优化空间，本项目在固定 155 tap、18 bit Q16 系数不变的条件下，对量化后绝对值较小的系数进行了裁剪搜索。

未裁剪的 4× FIR 在两种模式下的性能为：

| 模式 | 通带 ±纹波 | 阻带衰减 | 是否满足 |
|---|---:|---:|---|
| 44.1 kHz → 176.4 kHz | 0.01620230 dB | 70.23625475 dB | 是 |
| 48 kHz → 192 kHz | 0.01620252 dB | 70.23625475 dB | 是 |

搜索结果表明，当裁剪阈值 `prune_thr = 1` 时，虽然完整非零系数仅由 155 个减少至 153 个，但两种模式均不能继续满足指标。因此，4× FIR 前级对小系数裁剪非常敏感，不适合在固定 155 tap 条件下进行稀疏化处理。

最终设计保留 4× FIR 前级的完整 155 tap 高精度结构，并将主要优化集中在后级 2× FIR 的字长压缩和累加器位宽优化上。

## 11. 演示步骤

### 11.1 主验证：正弦 ROM + AD9708

1. 下载 `sine_rom_dual_family_ad9708_ok.bit`。

## 12. 注意事项

1. VS1053 支路仅用于听感辅助，不作为 4× / 8× / 128× 输出采样率的主验证证据。
2. 4× / 8× / 128× 输出采样率验证应以 AD9708 + 示波器结果为准。
3. 功耗报告置信度为 Medium，若需要更高精度功耗结果，可进一步导入 SAIF/VCD 活动文件。
4. 4× FIR 前级过渡带较窄，固定 155 tap 条件下对小系数裁剪非常敏感，因此最终设计不对 4× FIR 进行稀疏化裁剪。
5. 若现场只允许 FPGA 开发平台与示波器，优先演示 AD9708 主验证支路；VS1053 为额外辅助展示。

## 14. 最终结论

本项目已完成高阶数字插值滤波器的 MATLAB 建模、RTL 实现、功能验证、FPGA 板级验证和实现后资源 / 功耗 / 时序评估。系统能够处理 24 bit signed、44.1 kHz / 48 kHz 音频采样输入，并输出 4×、8×、128× 插值结果。板级主验证通过 AD9708 + 示波器完成，辅助听感展示通过 VS1053 A/B 对比完成。

在创新指标方面，项目完成了基于多级插值链路的分级结构优化和后级 2× FIR 字长优化。最终 acc_opt 版本采用 4× FIR 高精度前级与 2× FIR 低字长后级相结合的结构，在保证频响性能、线性相位、时序收敛和板级输出正常的前提下，将 Slice LUTs 从 17572 降低至 14852，将 DSP48E1 从 6 个降低至 1 个，将 CARRY4 从 4043 降低至 3383，证明该结构与字长优化策略能够有效降低 FPGA 资源消耗。
