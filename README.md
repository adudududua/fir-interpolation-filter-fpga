# 高阶数字插值滤波器设计与验证

## 1. 项目简介

本项目面向“数字设计域：高阶数字插值滤波器设计与验证”赛题，完成了适用于高保真音频重构场景的 FPGA 原型系统设计。系统围绕 24 bit 有符号音频采样信号，完成 MATLAB 建模、RTL 编码、功能仿真、板级验证、资源利用率评估、功耗评估与时序分析。

项目核心目标是在 FPGA 内部实现多倍率 FIR 插值滤波链路，支持 44.1 kHz / 48 kHz 两类音频采样率家族，并输出 4×、8×、128× 三档插值结果。板级主验证采用 AD9708 高速 DAC 与示波器完成；辅助听感展示采用 VS1053 音频解码模块完成 A/B 对比播放，用于直观展示保持重构与 FIR 插值重构之间的听感差异。

## 2. 赛题规格对应关系

### 2.1 输入规格

系统输入信号满足赛题要求：

| 项目 | 参数 |
|---|---|
| 输入数据格式 | 24 bit signed |
| 输入采样率 | 44.1 kHz / 48 kHz |

### 2.2 输出规格

系统支持以下输出采样率：

| 采样率家族 | 插值倍率 | 输出采样率 |
|---|---:|---:|
| 44.1 kHz | 4× | 176.4 kHz |
| 44.1 kHz | 8× | 352.8 kHz |
| 44.1 kHz | 128× | 5.6448 MHz |
| 48 kHz | 4× | 192 kHz |
| 48 kHz | 8× | 384 kHz |
| 48 kHz | 128× | 6.144 MHz |

### 2.3 核心指标

本项目以 FIR 低通插值滤波器作为核心结构，设计目标包括：

- 通带范围：10 Hz ~ 20 kHz
- 通带纹波：≤ ±0.05 dB
- 阻带衰减：≥ 70 dB
- 相位响应：严格线性相位
- FPGA 板级直观验证：AD9708 + 示波器
- 辅助听感展示：VS1053 A/B 对比

## 3. 系统总体结构

本项目采用“两条验证线路”：

```text
线路 1：主验证线路
正弦 ROM / 音频 PCM ROM
        ↓
24 bit signed 输入采样
        ↓
FPGA FIR 插值链
        ↓
4× / 8× / 128× 插值输出
        ↓
AD9708 高速 DAC
        ↓
示波器观察 dac_clk 与模拟输出波形

线路 2：辅助听感线路
48 kHz / 24 bit signed 音频 ROM
        ↓
FPGA 内部 A/B 实时处理
        ├─ A 路：保持重构
        └─ B 路：FIR 插值重构
        ↓
VS1053 播放 48 kHz / 16 bit PCM
        ↓
耳机 / 喇叭听感对比
```

其中 AD9708 + 示波器为赛题主验证支路；VS1053 为辅助听感展示支路，不作为 4× / 8× / 128× 高采样率输出的主验证证据。

## 6. 主要工程版本说明

### 6.1 `sine_rom_dual_family_ad9708_ok.bit`

用途：正弦 ROM 输入下的主验证版本。

该版本用于验证 FPGA 内部插值链路、采样率家族切换和插值倍率切换是否正确。输入为内部正弦 ROM，输出经 AD9708 接入示波器。

拨码功能：

| sw2 | sw1 sw0 | 采样率家族 | 插值倍率 | 理论输出采样率 |
|---|---|---|---:|---:|
| 0 | 00 | 44.1 kHz | 4× | 176.4 kHz |
| 0 | 01 | 44.1 kHz | 8× | 352.8 kHz |
| 0 | 10 / 11 | 44.1 kHz | 128× | 5.6448 MHz |
| 1 | 00 | 48 kHz | 4× | 192 kHz |
| 1 | 01 | 48 kHz | 8× | 384 kHz |
| 1 | 10 / 11 | 48 kHz | 128× | 6.144 MHz |

### 6.2 `audio_pcm_dual_family_ad9708_ok.bit`

用途：音频 PCM ROM 输入下的主验证版本。

该版本为最终主验证工程。输入源为 24 bit signed 音频 PCM ROM，系统仍支持 44.1 kHz / 48 kHz 两类采样率家族和 4× / 8× / 128× 三档插值输出。该版本用于导出资源、功耗与时序报告。

### 6.3 `vs1053_fpga_48k_interp_ab_ok.bit`

用途：辅助听感展示。

该版本读取 48 kHz / 24 bit signed 音频 ROM，在 FPGA 内部实时生成两路音频：

- `sw0 = 0`：A 路，保持重构，声音更粗糙、更尖锐；
- `sw0 = 1`：B 路，FIR 插值重构，声音更平滑、更柔和。

VS1053 只负责播放 FPGA 输出的 48 kHz / 16 bit PCM 音频流，A/B 差异由 FPGA 内部实时产生。

## 10. 演示步骤

### 10.1 主验证：正弦 ROM + AD9708

1. 下载 `sine_rom_dual_family_ad9708_ok.bit`。
2. 示波器 CH1 接 AD9708 模拟输出。
3. 示波器 CH2 接 `dac_clk`。
4. 切换 `sw2` 选择 44.1 kHz / 48 kHz 家族。
5. 切换 `sw1 sw0` 选择 4× / 8× / 128× 插值倍率。
6. 观察 `dac_clk` 是否对应理论输出采样率。

### 10.2 主验证：音频 PCM ROM + AD9708

1. 下载 `audio_pcm_dual_family_ad9708_ok.bit`。
2. 输入源为 24 bit signed 音频 PCM ROM。
3. 通过示波器观察复杂音频波形与 `dac_clk`。
4. 验证真实音频输入下系统仍能完成采样率家族和插值倍率切换。

### 10.3 辅助听感：VS1053 A/B

1. 下载 `vs1053_fpga_48k_interp_ab_ok.bit`。
2. 连接 VS1053 模块耳机输出。
3. `sw0 = 0`，播放保持重构版本。
4. `sw0 = 1`，播放 FIR 插值重构版本。
5. 对比 A/B 听感差异。

## 11. 创新点与优化说明

1. **多采样率家族支持**  
   系统同时支持 44.1 kHz 与 48 kHz 两类音频输入采样率家族，并可输出 4×、8×、128× 三档插值结果。

2. **对称 FIR 结构优化**  
   FIR 核心采用对称结构实现，利用线性相位 FIR 系数对称性减少乘法器资源开销。最终 DSP48E1 仅使用 6 个，占 XC7A35T 可用 DSP 资源的 6.67%。

3. **主验证与听感辅助分离**  
   AD9708 + 示波器用于证明采样率与板级输出正确性；VS1053 用于听感辅助展示。两条验证线路分工明确，避免将 VS1053 误作为 128× 高采样率输出验证手段。

4. **FPGA 内部实时 A/B 听感生成**  
   VS1053 听感支路中，A/B 两路来自同一段 48 kHz / 24 bit 输入 ROM。A 路为 FPGA 内部保持重构，B 路为 FPGA 内部 FIR 插值重构，差异不是由 MATLAB 预处理生成，而是在 FPGA 内部实时产生。

5. **工程化文件管理**  
   所有关键 Verilog 文件均采用统一文件头格式，包含文件名、模块名、功能简述、默认配置、作者、日期、版本与修订记录，便于答辩展示与后续维护。

---

## 12. 注意事项

1. VS1053 支路仅用于听感辅助，不作为 4× / 8× / 128× 输出采样率的主验证证据。
2. 4× / 8× / 128× 输出采样率验证应以 AD9708 + 示波器结果为准。
3. 功耗报告置信度为 Medium，若需要更高精度功耗结果，可进一步导入 SAIF/VCD 活动文件。
4. 当前主工程 LUT 占用率较高，主要原因是系统同时保留双采样率家族、多倍率插值链路以及板级演示逻辑。
5. 若现场只允许 FPGA 开发平台与示波器，优先演示 AD9708 主验证支路；VS1053 为额外辅助展示。
