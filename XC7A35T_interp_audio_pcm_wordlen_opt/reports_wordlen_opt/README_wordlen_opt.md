# 高阶数字插值滤波器设计与 FPGA 验证

## 1. 项目简介

本项目面向“高阶数字插值滤波器设计与验证”赛题，完成了基于 FPGA 的音频插值滤波系统设计、仿真、实现与板级验证。系统以 24 bit signed 音频 PCM 数据作为输入，支持 44.1kHz / 48kHz 两个音频采样率家族，并通过多级 FIR 插值结构实现 4×、8×、128× 输出采样率。

系统最终在 Artix-7 XC7A35T FPGA 平台上完成实现，并通过 AD9708 DAC 输出至示波器进行板级验证。为了满足赛题中的创新指标要求，本项目在基准版本基础上进一步完成了后级 2× FIR 的字长优化和分级结构优化，在保持滤波性能与板级输出功能正常的前提下，有效降低了 FPGA 资源消耗。

## 3. 系统总体结构

系统采用多级插值结构，由前级 4× FIR 插值器和多个后级 2× FIR 插值器级联构成。整体数据流如下：

```text
24 bit signed PCM 输入
        ↓
4× FIR 插值
        ↓
8× 输出 / 继续插值
        ↓
2× FIR 插值级联
        ↓
16× / 32× / 64× / 128×
        ↓
AD9708 DAC 输出
        ↓
示波器观察波形与 dac_clk 频率
```

系统通过拨码开关选择采样率家族和输出倍率：

| 开关                  | 功能              |
| ------------------- | --------------- |
| `sw2 = 0`           | 44.1kHz 输入采样率家族 |
| `sw2 = 1`           | 48kHz 输入采样率家族   |
| `sw1 sw0 = 00`      | 4× 输出           |
| `sw1 sw0 = 01`      | 8× 输出           |
| `sw1 sw0 = 10 / 11` | 128× 输出         |

## 5. 创新指标 1：结构 / 算法优化

### 5.1 分级插值结构

本项目没有采用单级直接 128× 插值结构，而是采用 4× FIR 与多个 2× FIR 级联的多级插值方案。该结构将高倍率插值任务分解为多个较低复杂度的插值阶段，使系统能够灵活支持 4×、8×、128× 多倍率输出。

### 5.2 基于频率选择性的分级优化策略

在基准版本中，前级 4× FIR 与后级 2× FIR 采用统一的高精度定点参数。经过分析可知，不同插值级的滤波压力并不相同：

* 前级 4× FIR 需要完成 44.1kHz / 48kHz 输入音频到 176.4kHz / 192kHz 的关键插值滤波，过渡带较窄，因此对滤波器阶数和系数精度要求较高。
* 后级 2× FIR 的输入采样率已经较高，过渡带明显更宽，因此可以采用更低的系数字长和更小的累加器位宽。

因此，优化版采用“前级保持高精度，后级压缩字长”的分级结构优化策略：

```text
4× 前级 FIR：
    保持原始高精度设计
    COEFF_W = 18
    FRAC_W  = 16
    ACC_W   = 56

后级 2× FIR：
    采用字长优化设计
    COEFF_W = 14
    FRAC_W  = 12
    ACC_W   = 48
```

这种结构优化避免了所有滤波级统一使用保守高字长参数，降低了后级重复插值链路的硬件资源开销。

## 6. 创新指标 2：字长优化策略

### 6.1 字长优化方法

为了进一步降低硬件资源消耗，本项目建立了 MATLAB 定点字长搜索流程，对后级 2× FIR 的系数字长、小数位宽和小系数裁剪阈值进行联合搜索。搜索对象包括：

```text
FRAC_W = 16 / 15 / 14 / 13 / 12
COEFF_W = FRAC_W + 2
PRUNE_THR = 0 / 1 / 2 / 4 / 8 / 16
```

搜索目标是在满足滤波性能指标的前提下，选择更低的系数字长和更少的有效系数。最终后级 2× FIR 选取：

```text
COEFF_W = 14
FRAC_W  = 12
ACC_W   = 48
```

### 6.2 后级 2× FIR 优化后频响性能

优化后的 2× FIR 同时适用于 176.4kHz → 352.8kHz 和 192kHz → 384kHz 两种模式，MATLAB 验证结果如下：

| 模式                  |       通带峰峰纹波 |       通带 ±纹波 |          阻带衰减 |        群延迟 | 是否满足 |
| ------------------- | -----------: | -----------: | ------------: | ---------: | ---- |
| 176.4kHz → 352.8kHz | 0.00201752dB | 0.00100876dB | 80.42743153dB | 14 samples | 是    |
| 192kHz → 384kHz     | 0.00132787dB | 0.00066393dB | 83.21212199dB | 14 samples | 是    |

该结果说明，后级 2× FIR 在系数由 18bit Q16 降低到 14bit Q12 后，仍然满足通带纹波、阻带衰减和线性相位要求。

### 6.3 字长优化前后对比

| 滤波级    | 基准版本                | 优化版本                |
| ------ | ------------------- | ------------------- |
| 4× FIR | 18bit Q16，56bit 累加器 | 保持不变                |
| 2× FIR | 18bit Q16，56bit 累加器 | 14bit Q12，48bit 累加器 |
| 优化策略   | 全链路统一保守字长           | 按滤波级频率选择性分级优化       |
| 优化目的   | 保证功能正确              | 在保证性能的前提下降低资源       |

## 12. 优化版本与创新指标完成情况

### 12.1 最终优化版本说明

在基准版本 `audio_pcm_dual_family_ad9708_ok` 的基础上，本项目进一步完成了字长优化、累加器位宽优化以及前级 4× FIR 稀疏化可行性分析。最终确定 `acc_opt` 版本作为优化版工程。

最终优化版参数如下：

| 滤波级       |   系数字长 | 小数位宽 |  累加器位宽 | tap 数 | 优化策略             |
| --------- | -----: | ---: | -----: | ----: | ---------------- |
| 4× FIR 前级 | 18 bit |  Q16 | 52 bit |   155 | 保持高精度结构，仅压缩累加器位宽 |
| 2× FIR 后级 | 14 bit |  Q12 | 45 bit |    29 | 系数字长优化 + 累加器位宽优化 |

其中，4× FIR 前级负责最窄过渡带滤波，对阻带衰减和系数精度要求较高，因此保留 18 bit Q16 系数；后级 2× FIR 的过渡带较宽，适合进行低字长优化，因此将系数由原始 18 bit Q16 优化为 14 bit Q12，并将累加器位宽进一步压缩至 45 bit。

---

### 12.2 字长优化策略

本项目针对后级 2× FIR 建立了 MATLAB 定点字长搜索流程，对 `COEFF_W`、`FRAC_W` 和小系数裁剪阈值进行联合搜索。搜索后得到的优化方案为：

```text
COEFF_W = 14
FRAC_W  = 12
ACC_W   = 45
NTAPS   = 29
```

优化后的 2× FIR 在两种工作模式下均满足设计指标：

| 模式                  |        通带峰峰纹波 |        通带 ±纹波 |           阻带衰减 |        群延迟 | 是否满足 |
| ------------------- | ------------: | ------------: | -------------: | ---------: | ---- |
| 176.4kHz → 352.8kHz | 0.00201752 dB | 0.00100876 dB | 80.42743153 dB | 14 samples | 是    |
| 192kHz → 384kHz     | 0.00132787 dB | 0.00066393 dB | 83.21212199 dB | 14 samples | 是    |

该结果说明，后级 2× FIR 即使从 18 bit Q16 降低到 14 bit Q12，仍能满足通带纹波、阻带衰减和严格线性相位要求。

---

### 12.3 累加器位宽优化

在完成后级 2× FIR 系数字长优化后，本项目进一步对 FIR 内部累加器位宽进行压缩。考虑到不同滤波级的乘加动态范围不同，最终采用分级累加器位宽：

```text
4× FIR：ACC_W = 56 → 52
2× FIR：ACC_W = 48 → 45
```

该优化不改变 FIR 系数和频率响应，只减少内部乘加链路、舍入饱和模块和比较逻辑的数据通路宽度。实现结果表明，累加器位宽优化后系统仍然满足时序要求，并能够正常完成板级 DAC 输出。

---

### 12.4 4× FIR 稀疏裁剪可行性分析

为了进一步考察前级 4× FIR 是否存在稀疏化优化空间，本项目在固定 155 tap、18 bit Q16 系数不变的条件下，对量化后绝对值较小的系数进行了裁剪搜索。

未裁剪的 4× FIR 在两种模式下的性能为：

| 模式                 |        通带 ±纹波 |           阻带衰减 | 是否满足 |
| ------------------ | ------------: | -------------: | ---- |
| 44.1kHz → 176.4kHz | 0.01620230 dB | 70.23625475 dB | 是    |
| 48kHz → 192kHz     | 0.01620252 dB | 70.23625475 dB | 是    |

搜索结果表明，当裁剪阈值 `prune_thr = 1` 时，虽然完整非零系数仅由 155 个减少至 153 个，但两种模式均不能继续满足指标。因此，4× FIR 前级对小系数裁剪非常敏感，不适合在固定 155 tap 条件下进行稀疏化处理。

最终设计保留 4× FIR 前级的完整 155 tap 高精度结构，并将主要优化集中在后级 2× FIR 的字长压缩和累加器位宽优化上。

### 12.8 创新指标总结

本项目围绕赛题创新指标完成了以下工作：

1. **结构 / 算法优化**
   系统采用 4× FIR + 多级 2× FIR 的分级插值结构，并根据不同滤波级的过渡带宽度和频率选择性要求采用不同优化策略。4× FIR 前级保持高精度结构，2× FIR 后级采用低字长优化，从而避免全链路统一使用保守高字长设计。

2. **字长优化策略**
   项目通过 MATLAB 定点字长搜索确定后级 2× FIR 的最优参数，将系数字长由 18 bit Q16 降低至 14 bit Q12，并进一步将累加器位宽压缩至 45 bit。优化后仍满足滤波器频响、线性相位、时序和板级验证要求。

最终优化版相较基准版本，Slice LUTs 从 17572 降低至 14852，DSP48E1 从 6 个降低至 1 个，CARRY4 从 4043 降低至 3383，证明该结构与字长优化策略能够有效降低 FPGA 资源消耗。
